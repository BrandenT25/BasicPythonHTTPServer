Custom HTTP Server

This is a custom HTTP server built from scratch using Python's socket library. No frameworks like Flask or Django - just raw socket programming to handle HTTP requests.

Features

  • Built using only Python sockets
  • Handles GET and POST requests
  • Custom routing system
  • Request logging with timestamps and IP addresses
  • Dockerized deployment
  • Running on AWS EC2
  • Infrastructure managed with Terraform
  • Testing with Pytest and Bash Scripts
  • CI/CD with GitHub Actions
  • Monitoring with Prometheus

How It Works

The server creates a TCP socket, binds to port 8000, and listens for incoming connections. When a client connects, it reads the HTTP request, parses the method and path, routes to the appropriate handler, and sends back an HTTP response.

Tech Stack

Python | Docker | AWS EC2 | Terraform | GitHub Actions | Prometheus | Linux

Check out /metrics for Prometheus metrics or visit the about page to learn more.
